# 🤖 Auth0 Advanced AI & RAG Bot

Implementation of Auth0 FGA, Token Vault, and Context-Preserving Assistant for MLH GenAI Hack Week 2025.

## Challenges Covered
1. **Privacy-Aware RAG Bot**: Fine-grained access control (FGA) for AI queries.
2. **The Veto Agent**: Token vaulting and asynchronous authorization.
3. **Context-Preserving Assistant**: Full-stack identity and API access management.

## Tech Stack
- Auth0 Fine-Grained Authorization (FGA)
- Auth0 Actions & Hooks
- Node.js / Express
- JWT / OAuth 2.0
